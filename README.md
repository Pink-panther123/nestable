# Nestable

A production-ready Next.js website, built to deploy on Vercel.

## Run locally
```bash
npm install
npm run dev
```

## Deploy
1. Push this folder to a GitHub repo.
2. Import it at https://vercel.com/new and click Deploy. Vercel auto-detects Next.js.
